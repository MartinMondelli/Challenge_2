## Overview

This repository contains the code and analysis developed for **Challenge 2**, a collaborative project focused on data processing, descriptive statistics, and predictive modeling.

The main work is done in the notebook `GONZALEZ_MODELLI.ipynb`, which includes data importing, cleaning, exploratory analysis, modeling, and results.
